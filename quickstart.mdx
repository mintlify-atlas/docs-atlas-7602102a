This guide takes you from an empty HTML file to a fully styled page with working dark mode toggle. No build tools required.

<Steps>
  <Step title="Add the TSS script tag">
    Paste the following `<script>` tag into the `<head>` of your HTML file. TSS auto-initializes — no setup code needed.

    ```html index.html
    <script src="https://cdn.jsdelivr.net/npm/tss-css/dist/tss.min.js"></script>
    ```

    <Note>
      TSS reads your `localStorage` for a saved theme preference before the page renders, so there is no flash of the wrong theme on reload.
    </Note>
  </Step>

  <Step title="Write utility classes">
    Start adding utility classes to your HTML elements. TSS detects them automatically and generates the CSS at runtime.

    ```html
    <div class="flex flex-col items-center p-8 bg-base">
      <h1 class="text-2xl font-bold text-heading mb-4">Hello, TSS</h1>
      <p class="text-body max-w-2xl text-center">
        Zero-config utility CSS with automatic dark mode.
      </p>
    </div>
    ```
  </Step>

  <Step title="Add a dark mode toggle">
    Call `TSS.toggleDark()` from any button. TSS switches the theme and persists the choice to `localStorage`.

    ```html
    <button
      class="px-4 py-2 rounded-lg bg-accent text-primary font-semibold cursor-pointer transition"
      onclick="TSS.toggleDark()"
    >
      Toggle dark mode
    </button>
    ```
  </Step>

  <Step title="Open the file in a browser">
    Open your HTML file directly in a browser — no server needed. Click the button to switch between light and dark mode. Reload the page and TSS restores your last selection.
  </Step>
</Steps>

## Complete example

Copy this file, open it in any browser, and you have a working styled page with dark mode.

```html index.html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>TSS quickstart</title>
  <script src="https://cdn.jsdelivr.net/npm/tss-css/dist/tss.min.js"></script>
</head>
<body class="bg-base min-h-screen flex flex-col items-center justify-center p-8">

  <div class="bg-surface rounded-lg shadow-md p-8 max-w-2xl w-full">

    <div class="flex items-center justify-between mb-6">
      <h1 class="text-2xl font-bold text-heading">TSS quickstart</h1>
      <button
        class="px-4 py-2 rounded-lg bg-accent text-primary font-semibold cursor-pointer transition"
        onclick="TSS.toggleDark()"
      >
        Toggle dark mode
      </button>
    </div>

    <p class="text-body mb-4">
      This page is styled entirely with TSS utility classes. No build step,
      no configuration file — just a script tag.
    </p>

    <div class="grid grid-cols-3 gap-4 mb-6">
      <div class="bg-secondary rounded-lg p-4 text-center">
        <p class="font-semibold text-heading">Zero config</p>
        <p class="text-subtle text-sm">One script tag</p>
      </div>
      <div class="bg-secondary rounded-lg p-4 text-center">
        <p class="font-semibold text-heading">Dark mode</p>
        <p class="text-subtle text-sm">Automatic tokens</p>
      </div>
      <div class="bg-secondary rounded-lg p-4 text-center">
        <p class="font-semibold text-heading">Runtime</p>
        <p class="text-subtle text-sm">No build step</p>
      </div>
    </div>

    <a
      href="https://github.com/"
      class="text-link underline"
    >
      View on GitHub
    </a>

  </div>

</body>
</html>
```

<Tip>
  TSS also exposes a `TSS.refresh()` method if you need to manually re-scan the DOM after programmatic content changes. In most cases the MutationObserver handles this for you.
</Tip>
